# Bankruptcy Prediction Analysis

This project analyzes corporate bankruptcy risk using financial data.

## Tools Used
- Excel / Python
- Statistical Analysis

## What I did
- Analyzed financial ratios (profitability, liquidity, leverage)
- Applied logistic regression and classification techniques
- Developed a risk classification system (low, medium, high)

## Dataset
- Taiwan Economic Journal dataset (1999–2009)
